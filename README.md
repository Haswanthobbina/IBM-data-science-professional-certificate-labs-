## 📂 Notebooks

### 📊 Data Visualization

| Notebook | Description |
|---|---|
| [Introduction to Matplotlib and Line Plots](./Introduction-to-Matplotlib-and-Line-Plots_.ipynb) | Foundations of data visualization using Matplotlib; creating and customizing line plots |
| [Area Plots, Histograms, and Bar Charts](./Area-Plots-Histograms-and-Bar-Charts_.ipynb) | Visualizing distributions and comparisons using area plots, histograms, and bar charts |
| [Pie Charts, Box Plots, Scatter Plots & Bubble Plots](./Exercise-Pie-Charts-Box-Plots-Scatter-Plots-and-Bubble-Plots.ipynb) | Exploring proportions, statistical spread, and relationships between variables |
| [Waffle Charts, Word Clouds & Regression Plots](./Waffle-Charts-Word-Clouds-and-Regression-Plots.ipynb) | Advanced visualizations: waffle charts for part-to-whole, word clouds for text data, and seaborn regression plots |
| [Plotting Directly with Matplotlib](./Plotting-directly-with-Matplotlib.jupyterlite.ipynb) | Deep-dive into Matplotlib's object-oriented API for fine-grained chart control |

## 🚀 Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/Haswanthobbina/IBM-data-science-professional-certificate-labs-.git
cd IBM-data-science-professional-certificate-labs-

# 2. Install dependencies
pip install pandas numpy matplotlib seaborn scikit-learn requests jupyterlab wordcloud pywaffle

# 3. Launch Jupyter
jupyter lab
```

Then open any `.ipynb` file from the file browser.
